# myothuko98/homebrew-tap

Homebrew tap for [myothuko98](https://github.com/myothuko98)'s tools.

## Install

```bash
brew install myothuko98/tap/git-explain
```

Or tap first then install:

```bash
brew tap myothuko98/tap
brew install git-explain
```

## Tools

| Formula | Description |
|---|---|
| `git-explain` | AI-powered git history explainer — understand *why* code changed |
